# 配置文件说明

本目录包含系统的可配置文件，支持动态扩展而无需修改代码。

## 配置文件列表

### 1. efficacy_keywords.json
**用途**：功效关键词配置
- 用于从"功能与主治"文本中提取功效标签
- 支持自定义添加新的功效关键词
- 导入脚本会自动加载此配置

**示例**：
```json
["发汗解表", "清热解毒", "活血化瘀", ...]
```

### 2. parsing_config.json
**用途**：药材名称解析配置
- **dose_patterns**: 剂量信息正则表达式
- **parentheses_patterns**: 括号内容移除规则
- **separators**: 药材名称分隔符
- **special_handlers**: 特殊处理规则

**配置项说明**：
| 配置项 | 说明 | 示例 |
|--------|------|------|
| dose_patterns | 匹配剂量信息的正则 | `\d+(\.\d+)?[gml斤两钱]*` |
| parentheses_patterns | 移除括号及其内容 | `\([^)]*\)` |
| separators | 药材分隔符列表 | `、`, `，`, `,` |
| special_handlers | 特殊字符处理 | `各.*$` 移除"各"后的内容 |

## 如何修改配置

1. 直接编辑对应的 `.json` 文件
2. 保存后无需重启服务（下次导入时自动生效）
3. 建议备份原始配置后再修改

## 注意事项

- 正则表达式需要正确转义
- JSON格式必须合法
- 配置错误会导致使用默认硬编码值
